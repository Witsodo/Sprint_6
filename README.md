# Sprint_6
Проект содержит автоматизированные UI-тесты для веб-приложения **Яндекс Самокат** с использованием Selenium WebDriver и Python.

## 📊 Покрытие тестами

### ❓ Тесты раздела FAQ
- Проверка 8 вопросов в аккордеоне:
  - Стоимость аренды
  - Несколько самокатов
  - Расчет времени аренды
  - Заказ на сегодня
  - Продление аренды
  - Зарядка самоката
  - Отмена заказа
  - Заказ за МКАД

### 🛵 Тесты оформления заказа
- Оформление через:
  - Верхнюю кнопку "Заказать"
  - Нижнюю кнопку "Заказать"
- Проверка полей формы:
  - Личные данные
  - Адрес доставки
  - Станция метро
  - Контактный телефон
  - Дата аренды
  - Срок аренды
  - Цвет самоката
  - Комментарий курьеру

### 🔗 Тесты навигации
- 🏠 Переход на главную через:
  - Логотип "Самокат"
  - Логотип "Яндекс" (открывает в новой вкладке)

## 🛠️ Технологический стек

- **Python 3.13**
- **Selenium WebDriver**
- **Pytest** (тестовый фреймворк)
- **GeckoDriver** (для автоматизации в Firefox)
- **Allure Report** (визуализация результатов)
